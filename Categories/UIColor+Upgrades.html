Overview

This UIColor category improves UIColor by adding four new class methods:

  • colorWithHex:
  • randomColor
  • colorFromSeed:
  • blueDetailColor

Tasks

  • + colorWithHex:

    Returns a UIColor instance with the hexadecimal color.
  • + randomColor

    Returns a new autoreleased UIColor instance with a random color, restricted to a subset of pastel colors.
  • + colorFromSeed:

    Returns a new autoreleased UIColor instance with a color obtained from the given seed.
  • + blueDetailColor

    Returns the blue color seen in detail labels in table views.
